proxmox 계정생성에서
pam 은 시스템에서도 생성해줘야 한다.

```bash title:hi
usermod -aG sudo dong-gu

vi /etc/ssh/sshd_config

PermitRootLogin no

PubkeyAuthentication yes

systemctl restart sshd

ssh-copy-id 192.168.20.100

# sudoers 파일 편집 
EDITOR=nano visudo 
# 다음 줄 추가 (기존 sudo 설정 아래에) 
dong-gu ALL=(ALL) NOPASSWD: ALL
```
